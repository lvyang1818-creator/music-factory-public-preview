# Responsive System

## Purpose

Responsive design must preserve the Music Factory brand experience across desktop, tablet, and mobile. The layout may change, but the editorial feeling must remain.

## Desktop

- Use generous whitespace.
- Use asymmetrical editorial layouts.
- Allow large display typography.
- Preserve right-side silence when the design calls for it.
- Avoid filling all available space.

## Tablet

- Reduce extreme offsets.
- Preserve hierarchy.
- Maintain readable line lengths.
- Keep imagery meaningful.
- Avoid compressed dashboard layouts.

## Mobile

- Stack content in strong editorial order.
- Keep tap targets usable.
- Preserve spacing.
- Avoid horizontal scroll.
- Make long words and titles wrap safely.
- Ensure images do not cover essential text.

## Breakpoint Guidance

| Name | Width | Rule |
| --- | --- | --- |
| Mobile | `< 760px` | Single column, large rhythm. |
| Tablet | `760px - 1100px` | Controlled two-column or stacked hybrid. |
| Desktop | `> 1100px` | Full editorial composition. |

## Text Rules

- Do not scale font size purely with viewport width in a way that becomes unstable.
- Check long English and Chinese strings.
- Avoid text overlap.
- Use line height and max-width to protect reading.

## Image Rules

- Preserve subject visibility on all breakpoints.
- Reconsider crop per breakpoint when needed.
- Do not let people or covers obscure text.

## Review Checklist

- Desktop has brand scale.
- Tablet has balanced hierarchy.
- Mobile is readable and calm.
- No horizontal scroll.
- No text overlap.
- No image obstruction.
