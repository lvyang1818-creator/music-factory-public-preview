# Music Factory Design System V1

## Purpose

This Design System defines the visual and interaction foundation for the Music Factory website. It is a documentation-only system for design decisions and should not be treated as a page implementation.

## Keywords

- Editorial
- Museum
- Luxury
- Archive
- Music Civilization
- Humanity
- Quiet
- Contemporary

## Forbidden Directions

- SaaS style.
- Startup style.
- Dashboard style.
- Technology-blue glow style.
- Generic AI-template style.
- Decoration without meaning.

## Files

| File | Role |
| --- | --- |
| `Color-System.md` | Defines palette, contrast, realm color behavior, and forbidden color directions. |
| `Typography-System.md` | Defines font roles, type scale, line height, and editorial type rules. |
| `Spacing-System.md` | Defines 8pt spacing rhythm and whitespace behavior. |
| `Grid-System.md` | Defines editorial grid, containers, museum navigation, and archive layout. |
| `Button-System.md` | Defines quiet CTA, link, button, hover, and accessibility behavior. |
| `Card-System.md` | Defines object-based card rules for artists, albums, works, realms, and archive entries. |
| `Section-System.md` | Defines section rhythm and page composition rules. |
| `Image-System.md` | Defines image, cover, crop, texture, and visual meaning rules. |
| `Motion-System.md` | Defines allowed motion, tokens, reveal behavior, and forbidden motion. |
| `Responsive-System.md` | Defines desktop, tablet, and mobile layout behavior. |
| `Design-Tokens.md` | Defines color, font, size, spacing, radius, shadow, and animation tokens. |

## Operating Rule

Future visual work should reference this design system before changing page UI. When a new visual decision does not fit this system, update the documentation first, then implement.

## Review Checklist

- Does the work feel editorial?
- Does it feel museum-quality?
- Does it avoid SaaS, startup, dashboard, and tech-glow aesthetics?
- Does it preserve Music Factory as a cultural brand?
- Does it use tokens and system rules instead of random one-off choices?
