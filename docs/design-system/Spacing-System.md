# Spacing System

## Purpose

Spacing defines the emotional pace of Music Factory. The system must feel Quiet, Editorial, Museum-like, and luxurious through restraint and rhythm.

## Base Grid

Music Factory uses an 8pt spacing system.

| Token | Value | Use |
| --- | --- | --- |
| `space.1` | `8px` | Fine gaps, minor offsets. |
| `space.2` | `16px` | Small component gaps. |
| `space.3` | `24px` | Standard internal spacing. |
| `space.4` | `32px` | Component rhythm. |
| `space.5` | `40px` | CTA and text group spacing. |
| `space.6` | `48px` | Small section separation. |
| `space.8` | `64px` | Medium section rhythm. |
| `space.10` | `80px` | Large editorial gaps. |
| `space.12` | `96px` | Major section spacing. |
| `space.16` | `128px` | Museum-scale whitespace. |

## Section Spacing

- Homepage and realm sections should generally use large vertical spacing.
- Hero sections may use full viewport rhythm.
- Dense UI panels are discouraged.
- Whitespace must create hierarchy, not simply push content apart.

## Internal Spacing

- Text groups should use controlled vertical gaps.
- Cards should not feel padded like SaaS feature cards.
- Navigation spacing must remain calm and readable.
- Archive rows may be denser, but still need breathing room.

## Responsive Spacing

- Desktop may use wide asymmetrical whitespace.
- Tablet should reduce extreme gaps but preserve rhythm.
- Mobile should stack with generous vertical spacing.
- Never collapse spacing until the page feels like an app dashboard.

## Forbidden

- Filling whitespace with decorative objects.
- Cramped card grids.
- Equal spacing everywhere without hierarchy.
- Dashboard density.
- Landing page section padding that feels generic.

## Review Checklist

- Does spacing create a sense of quiet value?
- Is there enough room for the brand to breathe?
- Does every large empty area feel intentional?
- Does mobile preserve the same emotional pace?
