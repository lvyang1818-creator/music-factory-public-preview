# Music Factory

> 音乐工厂（Music Factory）官方网站开发手册

---

# 项目简介

Music Factory 是一个集 AI 歌手、原创音乐、数字专辑、品牌宇宙、音乐内容、视觉艺术于一体的音乐品牌。

官方网站不仅用于展示作品，更承担品牌传播、内容运营、作品发布、歌手展示、用户沉淀等长期功能。

---

# 项目目标

建立一个具有国际审美的音乐品牌官方网站。

关键词：

- 高级克制
- Human Design
- Editorial
- 音乐文明
- 留白
- 国际化
- 去AI感
- 艺术策展

---

# 网站结构

Home（首页）

Artists（歌手）

Albums（专辑）

Songs（作品）

Universe（品牌宇宙）

About（关于我们）

Archive（品牌档案）

---

# 开发原则

所有页面保持统一设计语言。

所有组件统一维护。

所有页面遵循 UI 设计规范。

所有代码遵循开发规范。

所有图片遵循视觉规范。

所有更新记录写入 CHANGELOG。
