# 十里音 Visual Language

## Visual System

十里音 visual language should balance philosophical calm and contemporary sharpness.

## Color Direction

- Warm white.
- Charcoal.
- Deep gray.
- Restrained gold or red when meaningful.

## Image Direction

- Contemporary editorial portrait.
- Philosophical object imagery.
- Album artifact presence.
- Clean contrast.

## Motion Behavior

- Reveal.
- Mask.
- Controlled rhythm.

## Forbidden Visuals

- Cyberpunk.
- Generic future tech.
- Blue neon.
- Dashboard aesthetics.
