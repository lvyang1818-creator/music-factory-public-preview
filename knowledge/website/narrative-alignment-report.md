# Narrative Alignment Report

## Scope

This report checks whether Music Factory content expresses one civilization system.

## Master Narrative

Music Factory is A Civilization of Voices.

The website is a rendering surface for voices, memory, civilization, journey, and archive.

## Required Language

All systems should prefer:

- Voice.
- Memory.
- Civilization.
- Journey.
- Archive.
- World.
- Chapter.
- Record.
- Timeline.
- Entry.

## Forbidden Drift

Avoid:

- Product feature language.
- Dashboard language.
- File list language.
- Data visualization language.
- Player-only language.
- Generic landing page language.

## System Alignment

### Artists Universe

Artists are civilizations of identity, worldview, voice language, visual language, works, and archive entry.

### Sound Civilization Map

Sound Map is a living atlas of artist civilizations and their connections.

### Albums & Works

Works are civilization chapters.

Albums are memory collections.

### Archive System

Archive is the memory system for Works Timeline and Civilization Entries.

### Homepage

Homepage is the entry portal into the civilization system.

## Result

Pass.

The narrative layer is unified as a Civilization of Voices system.
