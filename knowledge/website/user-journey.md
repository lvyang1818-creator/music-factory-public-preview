# User Journey

## Purpose

The user journey defines how people should feel and move through Music Factory.

## Homepage Journey

1. Enter through silence, space, and brand atmosphere.
2. Understand that Music Factory is a cultural music brand.
3. Encounter the philosophy.
4. Explore realms.
5. Inspect latest releases.
6. Sense archive depth.
7. Leave with memory and curiosity.

## Realm Journey

1. Enter a world.
2. Understand its tone.
3. Meet the guardian.
4. Discover music works.
5. Read the philosophy.
6. Preview the archive.
7. Continue to another realm.

## Agent Rule

Every interface decision should support a calm, meaningful journey rather than a rushed conversion path.
