# Visual Language Audit

## Milestone

Milestone 15.6 - Product Experience Layer Unification.

## Purpose

This audit checks whether Artists, Sound Civilization Map, and Archive can operate as one visual system rather than three separate products.

## Required Visual Direction

Music Factory must preserve:

- Editorial Layout.
- Museum Grade Design.
- Quiet Luxury Aesthetic.
- Large Whitespace.
- Human-scale Composition.

The visual language must express A Civilization of Voices.

## Artists / Map / Archive Unity

### Artists

Artists Universe should feel like entering a personality world.

Required visual behavior:

- Strong identity.
- Human presence.
- Realm tone.
- Worldview and archive route.
- Slow narrative reading.

### Sound Map

Sound Civilization Map should feel like exploring civilization connections.

Required visual behavior:

- Atlas logic.
- Cultural passages.
- Map-like atmosphere without technical graph behavior.
- Artist worlds as civilizations.
- Archive routes as memory loops.

### Archive

Archive should feel like preserved civilization memory.

Required visual behavior:

- Quiet precision.
- Cultural context.
- Version and ownership clarity.
- Return routes to artist and map.
- Museum register, not data table density.

## Shared Visual Tokens

The audit confirms the existing system already provides:

- Shared display typography.
- Shared body and caption typography.
- Shared 8pt spacing.
- Shared warm paper / charcoal / dark gold direction.
- Shared slow reveal language.
- Shared editorial wrapper widths.

## Risk Areas

### Card Grid Drift

Works, albums, or artist entries may drift into generic card grids.

Fix:

Use editorial object presentation with ownership, memory, and route context.

### Archive Dashboard Drift

Archive may drift toward table or dashboard UI because it handles structured information.

Fix:

Use curated rows, museum labels, section rhythm, and return routes.

### Map Visualization Drift

Sound Map may drift toward relationship graph or data visualization.

Fix:

Use atlas, cultural route, and civilization language.

### Homepage Landing Page Drift

Homepage may drift toward a sales landing page.

Fix:

Keep Homepage as Portal to Civilization.

## Prohibited Visual Signals

- SaaS dashboard UI.
- Product listing layout.
- Generic card grid system.
- Data visualization UI.
- Technology glow.
- Complex particles.
- Game-like realm treatment.
- Player-first interface.

## Visual Acceptance Checklist

- Do Artists, Map, and Archive feel like layers of one civilization system?
- Does the page preserve editorial and museum quality?
- Does the page use large whitespace?
- Does the page avoid generic product cards?
- Does the page avoid dashboard density?
- Does the visual system support memory, voice, world, and journey?

## Audit Status

Status: Unified visual direction established.

Required future action: Any UI implementation after Milestone 15.6 must pass this audit before shipping.
