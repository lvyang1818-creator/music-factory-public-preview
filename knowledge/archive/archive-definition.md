# Archive Definition

## Definition

Archive is the memory system of Music Factory.

It preserves works, versions, albums, years, types, realms, artists, and cultural context.

## Archive Is Not

- A random list.
- A file dump.
- A playlist.
- A dashboard table.
- A storage folder without meaning.

## Archive Should Provide

- Continuity.
- Traceability.
- Cultural memory.
- Work relationships.
- Version history.
- Future retrieval.

## Archive Tone

Archive should feel quiet, precise, and museum-like.

## Agent Rule

When adding archive information, preserve context and relationship, not just filename and date.
