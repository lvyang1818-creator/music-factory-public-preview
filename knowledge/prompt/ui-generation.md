# UI Generation Prompt Knowledge

## Purpose

UI generation prompts must produce editorial brand interfaces, not SaaS dashboards.

## Prompt Principles

- Large whitespace.
- Museum pacing.
- Editorial grid.
- Quiet luxury.
- Warm white and charcoal.
- Restrained dark gold.
- Minimal interactions.

## Avoid

- Dashboard layout.
- Gradient CTA buttons.
- Glassmorphism panels.
- Tech blue glow.
- Feature-card stacks.
- Generic startup hero composition.

## Prompt Structure

1. Brand role.
2. Page role.
3. Layout rhythm.
4. Typography direction.
5. Color system.
6. Motion constraints.
7. Forbidden directions.

## Agent Rule

Any UI prompt must explicitly prohibit SaaS, startup, dashboard, and tech-glow aesthetics.
