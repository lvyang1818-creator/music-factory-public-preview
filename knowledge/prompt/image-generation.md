# Image Generation Prompt Knowledge

## Purpose

Image generation prompts must support the Music Factory visual world without creating generic AI aesthetics.

## Prompt Principles

- Editorial.
- Museum quality.
- Human-designed.
- Warm white space.
- Real material feeling.
- Album artifact presence.
- Cultural atmosphere.

## Avoid

- AI glow.
- Cyberpunk.
- Technology spheres.
- Random particles.
- Game fantasy.
- Over-decoration.
- Stock-photo emptiness.

## Prompt Structure

1. Subject.
2. Cultural context.
3. Material and light.
4. Composition.
5. Color restraint.
6. Negative constraints.

## Agent Rule

Do not generate images only because a page feels empty. Images must carry meaning.
