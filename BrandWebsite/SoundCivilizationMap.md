# Sound Civilization Map

## Positioning

Sound Civilization Map is a living atlas of voices, memories, and worlds.

It is not a data visualization, a relationship graph, an organization chart, a technology topology, or a dashboard.

It exists to show the overall worldview of Music Factory.

## Core Metaphor

Each artist is a civilization.

Each work is a chapter inside that civilization.

Each archive is a memory ruin, preserving traces of sound, image, story, and time.

## Page Structure

1. Sound Map Hero.
2. Voice Civilizations.
3. World Connections.
4. Archive Routes.
5. Journey Mode.
6. Future Expansion.

## Current Civilizations

### Spirit Realm

- Artist: 观无界
- Civilization logic: Pure Land, Buddhist chanting, silence, return, golden restraint.
- Current usable works: 无量光, 极乐, 四十八愿, 南无阿弥陀佛.
- Forbidden ownership: 无相, 心经, 道德经, 庄子.

### Future Realm

- Artist: 十里音
- Civilization logic: Eastern philosophy translated through contemporary pop, memory, future culture.
- Current works: 无相, 心经, 道德经, 庄子, 渊, 镜, 散, 无名, 姐姐出门了, 姐姐的爱很金贵.

### Nature Realm

- Artist: 梦梦
- Civilization logic: Nature, breath, softness, emotional weather, growth.
- Work rule: use placeholders until official works are defined.
- Forbidden ownership: Buddhist scripture, Daoist canon, classical philosophy, Spirit Realm works, Future Realm works.

## Connection Logic

Connections are emotional, cultural, sonic, philosophical, and narrative.

They must never be rendered as technical dependency lines, graph nodes, network topology, blockchain links, or database relationships.

## Archive Route Logic

The route is circular:

Civilization -> Artist -> Work -> Archive -> Sound Map.

The user should feel they are walking through a museum atlas, not filtering a database.

## Interaction Rules

- Hover reveal.
- Fade.
- Mask.
- Soft zoom.
- Slow motion.
- Subtle parallax.

Forbidden:

- Complex particles.
- Technology glow.
- Neural-network style lines.
- Large draggable maps.
- Zoomable map systems.
- Dashboard behavior.

## Visual Rules

The page should feel like a cultural institution, a literary world map, a museum guide, an ancient manuscript, and a quiet luxury editorial.

Use large whitespace, human scale, restrained typography, and meaningful visual hierarchy.

## Expansion Rule

The implementation must render civilizations from data.

Future fourth, fifth, sixth, and later artists must be addable by extending data, not by rewriting the layout.

# Sound Civilization Map V2

## Milestone

Milestone 19 - Sound Civilization Map V2.

## Definition

Sound Civilization Map V2 is A Living Civilization Atlas.

It upgrades the module from Sound Map as structure display to Sound Experience Map as immersive narrative.

It is not:

- A data map.
- A relationship graph.
- A technical network.
- A dashboard.
- A draggable topology.

It is:

A cinematic museum map for Music Factory's voice civilizations.

## Core Structure

The current atlas keeps three civilizations:

- 观无界 civilization space.
- 十里音 civilization space.
- 梦梦 civilization space.

Each civilization is not a node.

Each civilization is a space:

- Enterable.
- Immersive.
- Story-driven.
- Connected to artist, works, archive, and homepage.

## Interaction Rules

Allowed:

- Hover Reveal.
- Soft Zoom.
- Fade Transition.
- Mask Flow.

Forbidden:

- Graph UI.
- Data visualization.
- Technology network structure.
- Complex dragging.
- Zoomable map systems.

## User Path

The required V2 route:

Enter Map -> Choose Civilization -> Enter Space -> Watch Story -> Return Map.

## Data Rule

The implementation must derive civilization spaces from:

- Artists Knowledge Base.
- Works Catalog.
- `src/content/music-catalog.js`.

The page must not create independent UI-driven nodes.

## System Connections

Each civilization space must connect to:

- Artists Universe.
- Works.
- Archive.
- Homepage.

## Visual Keywords

- Cinematic Atlas.
- Museum Map.
- Cultural Landscape.
- Memory Geography.
- Editorial World Map.
