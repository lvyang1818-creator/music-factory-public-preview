# Visual System V3

## Position

Music Factory Visual System V3 upgrades the public experience from a functional system into a cinematic civilization experience.

The website is not presented as a product interface. It is presented as a cultural world made from voice, memory, archive, and time.

## Core Definition

Visual System V3 is:

A Cinematic Civilization Visual System.

It must express:

- Cinematic mood.
- Editorial rhythm.
- Museum-grade restraint.
- Human-scale composition.
- Slow interaction.
- Large whitespace.

It must avoid:

- Dashboard interface language.
- SaaS panels.
- Product listing layouts.
- Dense grid systems.
- Decorative interaction without narrative purpose.

## Typography

Typography becomes quieter and more cinematic.

- Display type is large, sparse, and slow.
- Body type is modest, legible, and calm.
- Captions are small, uppercase, and archival.
- Letter spacing stays neutral or lightly expanded only for captions.

## Spacing

Whitespace is a primary material.

- First screens should breathe.
- Sections must feel like rooms, not stacked modules.
- Content density must be reduced in visual presentation while keeping routes and data intact.

## Color

The palette is low-saturation and emotional:

- Deep ink.
- Warm paper.
- Aged gold.
- Clay.
- Moss.
- Smoke.
- Night plum.

No high-saturation tech colors.

## Layout

Layouts should feel editorial and cinematic:

- Asymmetric fields.
- Long horizontal routes.
- Portrait-led artist pages.
- Spatial map regions.
- Archive as a memory room.
- Works and albums as magazine features.

Avoid generic card grids. Repeated items can still exist, but they must read as story entries, archive traces, or civilization regions.

## Interaction

Interaction remains slow:

- Fade.
- Soft zoom.
- Mask reveal.
- Gentle hover.
- Page arrival.

No fast motion, no game mechanics, no particle effects.

## System Applications

Homepage:

- Cinematic entrance.
- Large title and breathing motion.
- Three civilization entry nodes.

Artists:

- Portrait dominant.
- Minimal copy on first impression.
- Works as story entries.

Sound Map:

- Civilization landscape.
- Region-based spaces.
- Entering a space, not clicking a graph point.

Works and Albums:

- Editorial magazine layout.
- Large feature rhythm.
- Work pages as immersive chapters.

Archive:

- Museum archive.
- Spatialized time flow.
- Memory room, not a list.
